## 二分法

1. 跟踪需要查找的部分，通过low和high两个点确定范围。
```
low=0
high=len(list)-1
```

2. 每次都检查中间的元素。
```
mid=(low+high)/2
guess=list[mid]
```

3. 如果猜的数字小了，就修改low；如果猜的数字大了，就修改high。
```
def binary_search(list,item):
	low=0
	high=len(list)-1
	
	while low<=high:
		mid=(low+high)
		guess=list[mid]
		if guess==item:
			return mid
		if guess>item:
			high=mid-1
		else:
			low=mid-1
	return None
```
